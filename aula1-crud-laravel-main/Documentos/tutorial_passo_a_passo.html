🚀 Tutorial Completo Laravel

Da configuração inicial ao teste de relacionamentos

1

Instalação e Configuração do Laravel

Este é o primeiro passo para qualquer projeto Laravel. Você precisa instalar o Laravel usando o Composer.

composer create-project laravel/laravel aula_1

Este comando:

  • Instala o Laravel e todas as suas dependências
  • Cria a estrutura de pastas do projeto
  • Configura o arquivo .env
ℹ️ Dica: O arquivo .env contém as configurações do seu projeto, como conexão com banco de dados, chave da aplicação, etc.
2

Configurar o Banco de Dados

Configure o arquivo .env com as credenciais do seu banco de dados.

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=aula_1
DB_USERNAME=root
DB_PASSWORD=
⚠️ Importante: Certifique-se de que o banco de dados já está criado no MySQL antes de rodar as migrações!
3

Criar a Migration da Tabela 'alunos'

Migrações são como controle de versão para seu banco de dados. Criam estrutura de tabelas de forma programática.

php artisan make:migration create_alunos_table

Isso cria um arquivo em database/migrations/ com o timestamp e nome da migração.

// Arquivo: database/migrations/2026_03_09_204417_create_alunos_table.php
public function up()
{
    Schema::create('alunos', function (Blueprint $table) {
        $table->id();
        $table->string('nome');
        $table->string('email')->unique();
        $table->date('data_nascimento');
        $table->timestamps();
    });
}
4

Criar o Model Aluno

O Model é a representação PHP da sua tabela. Permite manipular dados de forma orientada a objetos.

php artisan make:model Aluno
// Arquivo: app/Models/Aluno.php
namespace App\Models;

use Illuminate\Database\Eloquent\Model;

class Aluno extends Model
{
    protected $fillable = [
        'nome',
        'email',
        'data_nascimento',
    ];
}
ℹ️ $fillable: Define quais colunas podem ser preenchidas em massa (mass assignment).
5

Criar o Controller

Controllers tratam a lógica da requisição e controlam o fluxo entre rotas e modelos.

php artisan make:controller AlunoController
ℹ️ Convenção: Controllers no Laravel geralmente ficam em app/Http/Controllers/ e nomes são no singular + "Controller".
6

Implementar Método no Controller

Adicione o método testarRelacionamentos() que será responsável por testar os relacionamentos.

public function testarRelacionamentos()
{
    echo "<h1>Teste de Relacionamentos</h1>";
    echo "<hr>";

    // 1. Criar um aluno
    echo "<h2>1. Criando Aluno</h2>";
    $aluno = Aluno::create([
        'nome' => 'João Silva',
        'email' => 'joao.silva@email.com',
        'data_nascimento' => '2000-05-15',
    ]);
}
7

Configurar a Rota

Defina uma rota que conecte a URL ao seu controller e método.

// Arquivo: routes/web.php
use App\Http\Controllers\AlunoController;

Route::get('/alunos/teste-relacionamentos', [AlunoController::class, 'testarRelacionamentos']);
ℹ️ Estrutura da rota:
• Route::get() = método HTTP GET
• '/alunos/teste-relacionamentos' = URL
• [AlunoController::class, 'testarRelacionamentos'] = Controller e método
8

Executar as Migrações

Crie as tabelas no banco de dados executando as migrações.

php artisan migrate
✅ Sucesso! As migrações são executadas e a tabela 'alunos' é criada no banco de dados.
9

Testar a Aplicação

Acesse a URL em seu navegador para testar.

http://localhost/alunos/teste-relacionamentos
✅ Resultado esperado: Você verá a saída HTML mostrando:
- "Teste de Relacionamentos"
- "1. Criando Aluno"
- Dados do aluno criado

📊 Fluxo Geral do Laravel

URL
→
Route
→
Controller
→
Model
→
Database

Resumo do fluxo:

  1. URL: Usuário digita a URL no navegador
  2. Route: Laravel encontra a rota que corresponde
  3. Controller: Executa o método do controller
  4. Model: Interage com o banco de dados via Model
  5. Database: Dados são lidos/escritos no banco

📚 Conceitos-Chave Aprendidos

🔹 Migrações: Controle de versão do banco de dados

🔹 Models: Representação PHP das tabelas do banco

🔹 Controllers: Lógica da aplicação

🔹 Rotas: Mapeamento de URLs para ações

🔹 Eloquent ORM: Manipulação elegante de dados
